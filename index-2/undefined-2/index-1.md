# 인덱싱

> 인덱스(index) :일렬로 저장된 원소를 필요할 때 꺼내 사용할 수 있도록 붙인 태그입니다.
> 인덱싱(indexing) : 인덱스로 원소를 꺼내는 일
> 인덱스는 원소의 순서이고 0부터 시작합니다..

튜플의 인덱스를 [ ]에 전달해 인덱싱합니다.
```
for i in range(len(t_list1)):
    print(t_list1[i],end= ' ')
```  

역순은 음수의 인덱스 사용합니다.

```
t_list1[-1]
```
