## range로 튜플 만들기 

```
t_list5 =tuple(range(10))
print(t_list5)

t_list6 =tuple(range(5,15))
print(t_list6)

t_list7 =tuple(range(5,20,3)) # range(시작,끝,증가폭)
print(t_list7)

```
