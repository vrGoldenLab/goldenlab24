# 반복문 

## while 조건반복
조건이 참인 동안 계속 반복합니다.
while 문 내부에서 조건이 변경 하여 반복을 종료합니다.
```
a = 0
while a < 5:
    a += 1
    print('while 반복 수',a)
```
