# 홀수끼리,짝수끼리 모아모아 리스트  

## 프로그램 요구 사항  
* 1에서 n까지의 수를 리스트에 담습니다.( range 사용)
* index 1부터 2씩 건너 리스트 슬라이싱으로 짝수리스트를 만듭니다.
* index 0부터 2씩 건너 리스트 슬라이싱을 이용하여 홀수리스트를 만듭니다.
* 두 리스트를 더하여 결과를 출력해 확인합니다

> 참고> 인덱스 슬라이싱, 문자열 더하기
