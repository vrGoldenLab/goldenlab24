## 2.문자열

### 문자열은 텍스트의 기본 자료형

문자를 사슬(string)처럼 나영한 값, 자료형명 str    
변수명이나 함수명과 구분하기 위해 따옴표로 열고 닫아 생성합니다.  

문자열 내의 특수키  

\'   => '  
\"   => "  
\\   => \  
\n   => 줄바꿈(newline), enter  
\t   => 줄맞춤(intent),tap    

```
# '''로 문자열을 열고 닫을때
# \n없이 enter로 줄바꿈,\',\"없이 따옴표 사용 가능
print('''{
    마크다운 전환: ctrl + m m 
    코드로 전환: ctrl + m y 
    " 사용 가능  " '  '
}''')
```  
### 문자열에서 사용되는 함수  

* join : 문자열을 원하는 글자로 더한다.
* split : 문자열을 원하는 글자로 나눈다.
* strip : 문자열 앞뒤의 공백을 지운다. 문자열 내부의 공백은 지우지 않는다.
* replace : 문자열의 특정 글자를 다른 글자로 바꾼다.
* in : 문자열 안에 해당 글자가 있는지 확인
* not in : 문자열 안에 해당 글자가 없는지 확인 
* count : 특정 글자가 문자열에 몇번 사용되었는지 확인 
* find : 문자열에서 특정 글자의 위치정보 확인
* capitalize : 문자열의 첫 글자를 대문자로 변환
* lower : 모든 글자를 소문자로 변환
* upper: 모든 글자를 대문자로 변환
* starstwith : 해당 문자열이 특정 글자로 시작되는지 확인
* endswith : 해당 문자열이 특정 글자로 끝나는지 확인

### 문자열 포맷 코드
문자열 포매팅 예제에서는 대입해 넣는 자료형으로 정수와 문자열을 사용했으나 이 외에도 다양한 것을 대입할 수 있습니다.  
문자열 포맷 코드로는 다음과 같은 것이 있.

코드	설명
%s	문자열(String)
%c	문자 1개(character)
%d	정수(Integer)
%f	부동소수(floating-point)
%o	8진수
%x	16진수
%%	Literal % (문자 % 자체)
* %s 포맷 코드, 이 코드는 어떤 형태의 값이든 변환해 넣을 수 있습니다.  
`"I have %s apples" % 3` 출력결과 'I have 3 apples'
`"rate is %s" % 3.234`   출력결과 'rate is 3.234'
3을 문자열 안에 삽입하려면 %d를 사용하고, 3.234를 삽입하려면 %f를 사용해야 합니다.  
하지만 %s를 사용하면 자동으로 % 뒤에 있는 값을 문자열로 바꾸어 줍니다.
* 문자열 포맷 코드인 %d와 %가 같은 문자열 안에 존재하는 경우, %를 나타내려면 반드시 %%로 써야합니다.  
`"Error is %d%%." % 98`  

### 포맷 코드와 숫자 함께 사용하기  
1. 정렬과 공백  
`"%10s" % "hi"` 출력값  '        hi'
전체 길이가 10개인 문자열 공간에서 대입되는 값을 오른쪽으로 정렬하고 그 앞의 나머지는 공백.  
`"%-10sjane." % 'hi' ` 출력값 'hi        jane.'  
hi를 왼쪽으로 정렬하고 나머지는 공백.  

2. 소수점 표현하기  
`"%0.4f" % 3.42134234` 출력값 '3.4213'  
`"%10.4f" % 3.42134234` 출력값 '    3.4213'  

### format 함수를 사용한 포매팅  
1. 숫자 바로 대입하기  
`"I eat {0} apples".format(3)`

2. 문자열 바로 대입하기  
`"I eat {0} apples".format("five")`

3. 숫자 값을 가진 변수로 대입하기  
```
number = 3
"I eat {0} apples".format(number)
```

4. 2개 이상의 값 넣기  
2개 이상의 값을 넣을 경우 문자열의 {0}, {1}과 같은 인덱스 항목이 format 함수의 입력값으로 순서입니다.
```
number = 10
day = "three"
"I ate {0} apples. so I was sick for {1} days.".format(number, day)
```

5. 이름으로 넣기
{name} 형태를 사용할 경우 format 함수에는 반드시 name=value 와 같은 형태의 입력값이 있어야만 합니다.  
 
``` 
"I ate {number} apples. so I was sick for {day} days.".format(number=10, day=3)
```

6. 인덱스와 이름을 혼용해서 넣기  
```
"I ate {0} apples. so I was sick for {day} days.".format(10, day=3) 
```
7. 왼쪽 정렬
:<10 표현식을 사용하면 치환되는 문자열을 왼쪽으로 정렬하고 문자열의 총 자릿수를 10.
```
"{0:<10}".format("hi")
```

8. 오른쪽 정렬  
오른쪽 정렬은 :< 대신 :>을 사용하면 됩니다. 

```
"{0:>10}".format("hi")
```

9. 가운데 정렬
```
"{0:^10}".format("hi")
```

10. 공백 채우기   
정렬할 때 공백 문자 대신에 지정한 문자 값으로 채워 넣는 것도 가능하다. 채워 넣을 문자 값은 정렬 문자 <, >, ^ 바로 앞에 넣어야 합니다. 

```
"{0:=^10}".format("hi") #'====hi===='
```

```
"{0:!<10}".format("hi") #'hi!!!!!!!!'
```
11. 소수점 표현하기  
아래 예는 format 함수를 사용해 소수점을 4자리까지만 표현합니다. 앞에서 살펴보았던 표현식 0.4f를 그대로 사용한 것을 알 수 있습니다.  
```
 y = 3.42134234
"{0:0.4f}".format(y)
```
        위와 같이 자릿수를 10으로 맞출 수도 있습니다.
```
"{0:10.4f}".format(y)
```
12. { 또는 } 문자 표현하기  
format 함수를 사용해 문자열 포매팅을 할 경우 { }와 같은 중괄호를  문자 그대로 사용하고 싶은 경우에는 위 예의 {{ }}처럼 2개를 연속해서 사용.
```
 "{{ and }}".format() # '{ and }'
```

### f 문자열 포매팅
파이썬 3.6 버전부터는 f 문자열 포매팅 기능을 사용할 수 있습니다. 파이썬 3.6 미만 버전에서는 사용할 수 없는 기능입니다.  
다음과 같이 문자열 앞에 f 접두사를 붙이면 f 문자열 포매팅 기능을 사용할 수 있습니다.
1. f 문자열 포매팅은  변수 값을 참조할 수 있습니다. 
```
name = '홍길동'
age = 30
f'나의 이름은 {name}입니다. 나이는 {age}입니다.'
```
2. f 문자열 포매팅은 표현식을 지원 
※ 표현식이란 문자열 안에서 변수와 +, -와 같은 수식을 함께 사용하는 것을 말합니다.
```
age = 30
f'나는 내년이면 {age+1}살이 된다.'
```
3. f 문자열 포매팅에서 딕셔너리는 Key를 사용할 수 있습니다.

※ 딕셔너리는 Key와 Value라는 것을 한 쌍으로 갖는 자료형이다. 
```
d = {'name':'홍길동', 'age':30}
f'나의 이름은 {d["name"]}입니다. 나이는 {d["age"]}입니다.'
```
4. 정렬

`f'{"hi":<10}'  # 왼쪽 정렬` 'hi        '  
`f'{"hi":>10}'  # 오른쪽 정렬` '        hi'  
`f'{"hi":^10}'  # 가운데 정렬` '    hi    '    

5. 공백 채우기  
`f'{"hi":=^10}'  # 가운데 정렬하고 '=' 문자로 공백 채우기`
`f'{"hi":!<10}'  # 왼쪽 정렬하고 '!' 문자로 공백 채우기`

6. 소수점 
```
y = 3.42134234
f'{y:0.4f}'  # 소수점 4자리까지만 표현
```
```
f'{y:10.4f}'  # 소수점 4자리까지 표현하고 총 자리수를 10으로 맞춤 
```


7. { } 문자를 표시하려면 다음과 같이 두 개를 동시에 사용해야 합니다.

`f'{{ and }}'` 출력결과 '{ and }'
