# 코딩의 기초

# 프로그램의 기본 형식은 다음과 같습니다.

코딩 초보자들이 실제로 코드를 만들어 보려면 프로그래밍 언어를 다 외우는 것은 오히려 뭐가 뭔지 모르게 될 경우가 있습니다.

프로그래밍을 어떻게 하는지 배울 때, 언제나 "Hello World."를 가장 먼저 출력해본다는 전통이 있습니다. 
입력과 출력의 가장 단순한 예입니다.
``` 
 a= 'Hello World'
 print(a) 
 
```
